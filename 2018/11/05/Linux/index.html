Linux学习笔记

Linux基本命令

cd

用于切换当前目录

cd   /XXXX/XXXX    #切换到XXXX的XXXX目录下
cd   ./XXXX    #切换到当前目录下的XXXX目录下，’.’代表当前目录
cd   ../XXXX    #切换到上一层目录下的XXXX目录下，‘..’代表上层目录
cd   ..    #回到上层目录

ls

用于列出文件和目录

ls   -a    #列出所有文件，连同隐藏文件(开头为.的文件)
ls   -l    #列出所有文件，并显示出文件的属性权限数据
ls   -R    #连同子目录的内容一起列出，等于该目录下的所有文件都会显示出来
