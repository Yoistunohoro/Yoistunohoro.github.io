C语言笔记

常用的format字符串中的格式符

%d   对应参数是一个 int 类型
%f   对应参数是一个 float 类型
%ld   对应参数是一个 long int 类型
%lld   对应参数是一个 long long int 类型
%s   对应参数是一个 const char* 类型
%u   对应参数是一个 unsigned int 类型
%zu   对应参数是一个 size_t 类型
%td   对应的参数是一个 ptrdiff_t 类型
%x   对应的参数是一个 int 类型，不过会以16进制输出

int类型小细节

int类型对应的整数字面量可直接按照自然方式书写

八进制整数字面量以0打头

十六进制整数字面量以0x打头

无符号类型字面量书写可在字面量后面添加U或者u(一般书中为大写)
